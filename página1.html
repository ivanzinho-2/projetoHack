CSS Seletores id e class

Uma regra de CSS é um conjunto de “ordens” que se aplicam a uma tag selecionada com a finalidade de mudar sua aparência ou posição.

O seletor ID

O seletor .id define um elemento HTML que será modificado com CSS sem afetar outros elementos com a mesma tag. O seletor .id fornece um "nome próprio" para o elemento e só pode ser utilizado uma vez por página.

O seletor CLASS

O seletor class define um grupo de elementos HTML que recebem as mesmas regras do CSS e pode ser utilizado quantas vezes for necessário.
